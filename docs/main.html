main.main()¶

The main function

This function only accept command line input in the format of URL(link to the user history) pass it to get_history function to get history id and name, which are then pass into get_uset_workflow function to generate the user workflow stored as a dictionary. get_standard_workflow gets standard workflow as dictionary from URLs. These two dictionaries are passed into compare function, which returns a report in dictionary format. The report dictionary then goes into generate_report_file function to be turned into a JSON file.
