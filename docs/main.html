main.main()¶

The main function

This function serves as the driver and connection between all major parts of the program: It does four things:

  1. Take in 4 parameters via ArgumentParser.

  2. Call different fundtions to process the parameters, and to generate two workflows.

  3. Pass the two workflows to the compare function in history_compare.py

  4. Call generate_report_file function to output the final report.

Parameters:

  • history_url – the link to the user’s history, which is intended to be tested.

  • workflow_url – the link to the workflow, corresponding to the history, which serves as the standard.

  • apikey – a Galaxy API key obtained prehand

  • path – the desired system path to output the final report

Returns:

final report: a JSON file
