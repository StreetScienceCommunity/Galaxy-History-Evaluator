Installation Guide
=======================


**Requirements**

------------

**Python**

In order to use this tool, you need to have `Python` installed in your enviornment.

`ghevaluator` has been tested to be running with `Python` version higher or equal to `3.6`


------------

**pip**

You also need to have `pip` installed in order to install `ghevaluator`.

- If you already have `pip` installed, make sure it is up to date. You can try upgrade it via:

.. code-block:: bash

    $ python -m pip install -U pip

- If you don't have `pip` installed, follow ths official installation instruction here:
https://pip.pypa.io/en/stable/installation/

------------

**Install ghevaluator via pip**

Finally, to install `ghevaluator`, simply run the following command in the terminal:

.. code-block:: bash

    $ pip install ghevaluator
